# Comandos
git config --list             Mostra vários dados e o nome, email, senha, etc
git config --global user.name "nome"        Adiciona um nome ao --list
git config --global user.email "email"      Adiciona um email ao --list
git config --global user.password "senha"   Adiciona uma senha ao --list
git config --global --unset--all user = Tira o que você quiser, senhas, nomes, ou o email.
git clone (link)     Copia o repositório



git status         Mostra o estado atual do repositório da pasta e dos arquivos
git add            Adiciona um arquivo ou pasta feita manualmente ao repositório do GitHub
git commit -m      Registra um ponto no tempo do seu projeto
git push           Envia o que foi salvo no commit para o GitHub
git pull           Ele salva as mudanças que aconteceram
git init           Inicia o repositório
